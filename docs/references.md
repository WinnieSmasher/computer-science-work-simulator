# References

本仓库只借鉴公开项目的工程思想，不复制其代码。

- Google Cloud Online Boutique: https://github.com/GoogleCloudPlatform/microservices-demo
- Microsoft eShop: https://github.com/dotnet/eShop
- Backstage: https://github.com/backstage/backstage
- RealWorld App Spec: https://github.com/gothinkster/realworld

这些项目值得学习的地方：

- Online Boutique：微服务边界、商品/购物车/订单/支付/推荐/通知的业务覆盖面。
- eShop：参考架构、领域边界、容器化、本地开发体验。
- Backstage：服务目录、文档门户、平台工程意识。
- RealWorld：统一 API spec 以及多语言实现的教学方式。
