# 第 09 关 - 对接第三方接口

## 对应能力

- 能力简称：对接第三方接口
- 能力说明：读 API 文档、调接口、处理失败、鉴权、文件上传

## 后端模拟工作场景

运营要上传商品主图，团队决定先接一个模拟 OSS 文件服务。

## 这一关真正练什么

读 API 文档、调接口、处理失败、鉴权、文件上传

## 相关技术关键词

图片接收配置、分布式文件存储、枚举、OSS、瀑布流进阶、工具类

## 你接到的任务

- 实现文件上传签名申请接口。
- 封装模拟 OSS client，不把第三方 SDK 泄露到业务层。
- 处理上传失败、文件类型不支持和回调验签失败。

## 验收标准

- 业务层只依赖自定义接口。
- 错误码能区分本地校验失败和第三方失败。
- 文档写明阅读第三方文档的步骤。

## 学习提醒

这一关不是为了堆技术名词，而是训练“接到真实后端工作后如何理解、拆解、实现、验证、解释”的能力。代码能跑只是底线，能说明为什么这样做才算过关。
