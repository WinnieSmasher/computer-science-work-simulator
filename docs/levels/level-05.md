# 第 05 关 - 理解简易 C 端技术需求的能力

## 对应 PDF 能力

理解简易 C 端技术需求的能力

## 后端模拟工作场景

用户端商品列表要支持无限下拉，前端要求后端返回 hasMore/isEnd，且不能因为时间窗口和分页边界出错。

## 这一关真正练什么

实现用户端列表分页、状态过滤和边界判断。

## 相关技术关键词

isEnd、时间 API、常规 7 个接口、包装类与基本类型选型

## 你接到的任务

- 实现用户端商品列表分页，返回 hasMore 和 isEnd。
- 只展示已上架、未删除、在展示时间窗口内的商品。
- 补充 page/size 边界、空列表、最后一页测试。

## 验收标准

- 第一页、最后一页、越界页行为清晰。
- hasMore 与 isEnd 不互相矛盾。
- 能解释为什么 C 端分页不能只返回 list。

## 学习提醒

这一关不是为了堆技术名词，而是训练“接到真实后端工作后如何理解、拆解、实现、验证、解释”的能力。代码能跑只是底线，能说明为什么这样做才算过关。
