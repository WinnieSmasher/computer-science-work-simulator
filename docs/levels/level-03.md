# Level 03 - 简单技术需求落地

## 业务背景

NovaMart 的后台需要管理商品，用户端需要按关键词浏览已上架商品。

## 工作能力

把产品需求翻译成后端接口、字段校验、业务状态和测试。

## 任务

- 实现后台商品创建、详情、更新、逻辑删除。
- 实现用户端商品搜索：`GET /api/products`。
- 商品只在 `ACTIVE` 且未删除时出现在用户端。

## 验收标准

- 商品有名称、描述、价格、状态、标签 ID、删除标记。
- 搜索结果包含 `items`、`page`、`size`、`total`、`hasMore`、`isEnd`。
- 测试覆盖创建、搜索和逻辑删除。
