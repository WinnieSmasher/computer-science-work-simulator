# 第 18 关 - Redis 中间件 1

## 对应能力

- 能力简称：Redis 中间件 1
- 能力说明：Redis 部署、缓存场景、主从/哨兵/集群

## 后端模拟工作场景

商品详情和热点库存访问变多，需要引入 Redis 缓存。

## 这一关真正练什么

Redis 部署、缓存场景、主从/哨兵/集群

## 相关技术关键词

Redis 单机、主从、哨兵、集群、常见缓存场景

## 你接到的任务

- 为商品详情增加 Redis 缓存。
- 设计热点库存缓存 key 和 TTL。
- 处理缓存穿透、缓存击穿的最小方案。

## 验收标准

- 缓存命中和未命中行为可测试。
- 商品更新后缓存会失效。
- 能说明单机、主从、哨兵、集群差异。

## 学习提醒

这一关不是为了堆技术名词，而是训练“接到真实后端工作后如何理解、拆解、实现、验证、解释”的能力。代码能跑只是底线，能说明为什么这样做才算过关。
