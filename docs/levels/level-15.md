# 第 15 关 - 基本运维和线上实操的能力

## 对应 PDF 能力

基本运维和线上实操的能力

## 后端模拟工作场景

测试环境要部署 NovaMart 单体服务，导师只给一台 Linux 机器和一个域名。

## 这一关真正练什么

完成基本部署、反代、日志查看和远程排查。

## 相关技术关键词

ECS、Linux、CentOS、文件/文件夹、shell、管道、yum、rpm、源码安装、nginx、远程 debug

## 你接到的任务

- 编写部署说明和启动脚本。
- 配置 Nginx 反向代理到 Spring Boot。
- 整理常用日志查看、进程查看和端口排查命令。

## 验收标准

- 服务能在 Linux 上启动并被 Nginx 访问。
- 能根据日志定位启动失败原因。
- 不把密码写进仓库。

## 学习提醒

这一关不是为了堆技术名词，而是训练“接到真实后端工作后如何理解、拆解、实现、验证、解释”的能力。代码能跑只是底线，能说明为什么这样做才算过关。
