# 第 11 关 - 请求生命周期

## 对应能力

- 能力简称：请求生命周期
- 能力说明：浏览器/APP 到 Tomcat/Spring/DB 再返回的完整链路

## 后端模拟工作场景

一次订单预览请求变慢，导师要求你画出从 HTTP 到数据库再回到响应的完整链路。

## 这一关真正练什么

浏览器/APP 到 Tomcat/Spring/DB 再返回的完整链路

## 相关技术关键词

线程池、Bean、Tomcat、Servlet、Druid、数据库密码加密传输、锁常识

## 你接到的任务

- 为订单预览接口增加 traceId。
- 记录 Controller、Service、DB adapter 的耗时。
- 写出请求从客户端到返回的链路文档。

## 验收标准

- 能定位慢在参数校验、业务逻辑还是数据访问。
- 日志能按 traceId 串起来。
- 能解释 worker 线程、连接池和 Spring Bean 的角色。

## 学习提醒

这一关不是为了堆技术名词，而是训练“接到真实后端工作后如何理解、拆解、实现、验证、解释”的能力。代码能跑只是底线，能说明为什么这样做才算过关。
