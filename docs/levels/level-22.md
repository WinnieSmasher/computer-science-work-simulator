# 第 22 关 - 复杂 C 端需求

## 对应能力

- 能力简称：复杂 C 端需求
- 能力说明：更复杂的用户端业务方案，别迷信 PDF 里的“wp/isEnd”黑话

## 后端模拟工作场景

用户首页要做推荐瀑布流，要求曝光去重、分页稳定、客户端不能篡改游标。

## 这一关真正练什么

更复杂的用户端业务方案，别迷信 PDF 里的“wp/isEnd”黑话

## 相关技术关键词

opaque cursor、曝光去重、签名校验、加密和解密、复杂 C 端列表

## 你接到的任务

- 设计推荐流 cursor，不直接暴露数据库 offset。
- 记录已曝光商品，避免短时间重复出现。
- 为 cursor 增加签名或校验。

## 验收标准

- 刷新、下拉、重复请求行为稳定。
- 客户端伪造 cursor 会被拒绝。
- 能说明这种需求为什么不只是普通分页。

## 学习提醒

这一关不是为了堆技术名词，而是训练“接到真实后端工作后如何理解、拆解、实现、验证、解释”的能力。代码能跑只是底线，能说明为什么这样做才算过关。
