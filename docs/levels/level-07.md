# 第 07 关 - 理解代码规范和学习描述语言的能力

## 对应 PDF 能力

理解代码规范和学习描述语言的能力

## 后端模拟工作场景

团队要求把重复 CRUD 代码收敛到统一规范，并用描述语言生成部分模板。

## 这一关真正练什么

理解团队代码规范，引入 DTO/VO、分页规范和模板化生成思路。

## 相关技术关键词

MyBatis Plus、分页插件、FreeMarker、快速学习简单框架三步曲

## 你接到的任务

- 整理 Controller、Service、Response 的命名规则。
- 为一个后台配置模块生成基础 CRUD 模板。
- 写出模板生成后哪些代码必须人工检查。

## 验收标准

- 生成代码符合项目返回结构和异常规范。
- 没有把模板当成万能答案。
- 文档说明 DTO、VO、Entity 的职责区别。

## 学习提醒

这一关不是为了堆技术名词，而是训练“接到真实后端工作后如何理解、拆解、实现、验证、解释”的能力。代码能跑只是底线，能说明为什么这样做才算过关。
